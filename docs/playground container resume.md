# playground container resume

⏯️  Resume a container

| Attributes       | &nbsp;
|------------------|-------------
| Alias:           | unpause

## Usage

```bash
playground container resume [OPTIONS]
```

## Options

#### *--container, -c CONTAINER*

🐳 Container name  
  
🎓 Tip: you can pass multiple containers by specifying --container multiple times

| Attributes      | &nbsp;
|-----------------|-------------
| Repeatable:     |  ✓ Yes
| Default Value:  | connect
