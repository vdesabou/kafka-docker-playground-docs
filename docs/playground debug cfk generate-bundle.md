# playground debug cfk generate-bundle

⛑️ Generate a Support bundle  
  
see https://docs.confluent.io/operator/current/co-troubleshooting.html#support-bundle

## Usage

```bash
playground debug cfk generate-bundle [OPTIONS]
```

## Dependencies

#### *kubectl*

visit https://kubernetes.io/docs/tasks/tools/ to install

## Options

#### *--follow-logs-duration FOLLOW-LOGS-DURATION*

Max duration to follow logs

| Attributes      | &nbsp;
|-----------------|-------------
| Default Value:  | 0

## Examples

```bash
playground debug cfk generate-bundle
```
