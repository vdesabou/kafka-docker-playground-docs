# playground schema get-alias

🔯 Get subject alias  
  
CP 7.4.1 or higher is required to have this feature

## Usage

```bash
playground schema get-alias [OPTIONS]
```

## Options

#### *--alias ALIAS*

🔯 Alias name

| Attributes      | &nbsp;
|-----------------|-------------
| Required:       | ✓ Yes

#### *--verbose, -v*

🐞 Show command being ran.
